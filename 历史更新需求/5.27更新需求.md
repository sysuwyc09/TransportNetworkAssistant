# 总体需求
publicThread.py class OltKnowledgeThread(QThread) 类更新功能模块，用于添加千兆口、普通口空闲占用状体的知识库明细；
# 需求执行流程
1、从数据库读取《PON口数据集》表，标题包括（OLT网元(TEXT), 端口名称(TEXT), 端口状态(TEXT), PON口下挂用户数(INTEGER), 槽位号(INTEGER), 端口号(INTEGER), 板卡名称(TEXT), 板卡类型(TEXT), PON口类型(TEXT)）
2、根据数据处理，目标返回三列 OLT网元，PON口类型，端口情况
- 端口情况：OLT网元按PON口类型、槽位号、端口状态进行聚合，例如千兆口：2槽3、4、5口 占用，2槽6、7口 空闲，普通口：1槽1、2口 占用，1槽3、4口 空闲;
3、将处理结果和olt_df进行合并，并将端口情况加在详细情况之后
