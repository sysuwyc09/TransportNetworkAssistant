# **弱光整治**
- 限制读取表格的类型为xlsx，同时必须包含以下列：
    - PON口
- 弱光分析方法
```mermaid
graph TD;
    A[导入表格]-->B[数据清洗+分析光衰];
    B-->C[匹配最优方案];
    C-->D[筛选可行方案];
    D-->E[割接点分析];
    E-->F[生成结果表格];
```
- 杆路段落地分析（已有管道部分）
```mermaid
graph TD;
    A[导入表格、杆路段、电杆、管井]-->B[电杆周边200米管井数分析];
    B-->C[按数值粗略筛选200米，然后按公式计算距离，筛选200米内];
    B-->D[匹配有管井数>=1的值];
    D-->E[杆路段名称提取];
    E-->F[杆路段长度聚合];
```
